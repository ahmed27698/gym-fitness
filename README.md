# APEX FITNESS

Premium gym & fitness website built with Nuxt 4, GSAP, and Tailwind CSS.

## Features

- 20+ pages — home, shop, blog, classes, trainers, schedule, gallery, and more
- GSAP animations with ScrollTrigger
- 6 themes — Dark, Light, Neon, Gold, Crimson, Emerald
- Full Arabic RTL support with i18n
- Shopping cart, wishlist, and checkout flows
- BMI calculator, pricing plans, careers, and more
- Fully static — deployable to Netlify / Vercel / any CDN

## Tech Stack

- **Nuxt 4** — SSG (Static Site Generation)
- **Vue 3** + TypeScript
- **Tailwind CSS**
- **GSAP 3** + ScrollTrigger
- **@nuxtjs/i18n** — English & Arabic
- **@nuxt/image**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build & Deploy

```bash
npm run generate   # builds static output to .output/public
```

Deploy `.output/public` to any static host. A `netlify.toml` is included for one-click Netlify deployment.
